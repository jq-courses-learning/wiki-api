## Installation

```
Clone the repo
```

## Requirements
```
Node v10.15.3 
Nodemon 1.19.0 *just handy
```

## Usage

```
CD into the repo
Run => nodemon app.js or node app.js
```

## Description

```
This is a simple Wiki (api) project using Node.Js and EJS
The user can create/edit/delete articles using Postman for example.
```
